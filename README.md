# dotfiles

Personal configuration files for macOS setup.

## Usage

```bash
# Install dotfiles
./mac-setup.sh

# Setup development environment  
./dev-setup.sh
```

## iTerm2

```bash
# Export preferences
cp ~/Library/Preferences/com.googlecode.iterm2.plist .

# Import preferences
cp com.googlecode.iterm2.plist ~/Library/Preferences/
```
